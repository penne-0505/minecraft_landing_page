## Goal
Discord 参加を主 CTA とした一般公開 LP のビジュアルデザインとデザイントークンを作成する。モバイル優先で、既存 LP（Minecraft Server 🍀）のトーンを継承する。

## Deliverables
- Figma コンポーネントライブラリ（ボタン、カード、CTA バナー、FAQ、フッター）。
- 3 ブレークポイント（モバイル/タブレット/デスクトップ）のレイアウト案。
- カラー/タイポスケール/スペーシング/シャドウのトークン表（可能なら JSON 形式も）。
- 主要セクションのヒーロー画像指示（生成系を使う場合はプロンプト例も）。

## Must-Haves
- CTA: Discord 招待リンクを最上位に配置。ヘッダーにメンバーシップページへの導線を設置。
- モバイル最適化: ファーストビューが 1 スクロール内に CTA + ベネフィットを収まる構成。
- 法的リンクのプレースホルダ: 利用規約/プライバシー/特商法表記へのリンクエリア確保。
- アクセシビリティ: コントラスト比 WCAG AA 相当、フォーカス状態の可視化。

## Tone & References
- 既存 LP をデザイントークンの一次情報源とする。緑系アクセント + 柔らかい角丸。ロゴ指定なし。
- タイポはラウンド系 + 視認性重視。Illustration は必要最小限で可。

## Open Items for Designer
- 背景テクスチャ/パターンの提案可否。
- アイコンセットの統一（Lucide/Feather など）を使うか、独自セットを作るか。
- 画像アセットのホスティング場所（Cloudflare Images 等）の推奨。

- フィードバック反映: 2025-12-18
- デザイン確定: 2025-12-20
